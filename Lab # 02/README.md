# **Lab # 02 - Dart Programming & Data Structures**  

## **Overview**  
This lab introduces **fundamental Dart programming concepts**, covering essential **problem-solving techniques and data structures**. It includes implementations of **sorting algorithms, list manipulation, and stack operations**.  

## **Learning Objectives**  
✔ Develop a function to **find the largest number in a list**.  
✔ Implement **Merge Sort** for sorting a list of integers.  
✔ Create a **Stack data structure** with core functionalities (**push, pop, peek, size**).  

## **Tasks & Implementation**  

### **1️⃣ Finding the Largest Number in a List**  
🔹 Implemented a function to identify the **maximum value** in a given list.  
🔹 Utilized **iteration** to compare and update the largest number.  

### **2️⃣ Implementing Merge Sort Algorithm**  
🔹 Applied the **Divide and Conquer** approach to **sort a list of integers**.  
🔹 Divided the list into **two halves recursively** and merged them in sorted order.  

### **3️⃣ Implementing a Stack Data Structure**  
🔹 Developed a **generic Stack class** with core functionalities:  
✅ `push(T item)` → Adds an element.  
✅ `pop()` → Removes and returns the top element.  
✅ `peek()` → Returns the top element without removing it.  
✅ `isEmpty()` → Checks if the stack is empty.  
✅ `size()` → Returns the number of elements in the stack.
